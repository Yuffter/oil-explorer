# Oil Explorer

[oil.nvim](https://github.com/stevearc/oil.nvim) 風にファイル操作ができる Obsidian プラグインです。Vault 内のフォルダをテキストバッファとして開き、行を編集するだけでファイルの作成・リネーム・削除ができます。

## 特徴

- **フォルダをテキストとして編集**: フォルダを開くと、中身のファイル/フォルダ名が1行1エントリで表示されます。行を編集して保存すると、その差分がリネーム・作成・削除としてVaultに反映されます。
- **Vimキーバインド対応**: [`@replit/codemirror-vim`](https://github.com/replit/codemirror-vim) を使用し、oil.nvim と同様の操作感（`-` で親フォルダへ、`Enter` で開く、`:w` / `Ctrl+S` で保存）を実現しています。
- **Vim設定に追従**: デフォルトでは Obsidian 本体の「Vim key bindings」設定に自動追従します。コマンドから Oil Explorer 内だけ強制ON/OFFに切り替えることも可能です。
- **削除確認ダイアログ**: 行を消してファイル/フォルダを削除する操作には、誤操作防止のための確認モーダルが表示されます。

## 使い方

1. リボンアイコン（フォルダ）またはコマンドパレットから「現在のフォルダをOilで開く」を実行します。
2. 開いたビューには現在のフォルダ内のファイル・フォルダ名が一覧表示されます（フォルダ名は末尾に `/` が付きます）。
3. 行を編集して変更内容を指定します。
   - 行のテキストを書き換える → リネーム
   - 新しい行を追加する → 新規ファイル/フォルダ作成
   - 行を削除する → ファイル/フォルダ削除（確認ダイアログあり）
4. `Ctrl/Cmd + S`（Vimモードでは `:w` や `Ctrl+S`）で変更を確定します。
5. `Ctrl/Cmd + Enter`（Vimモードでは `Enter`）でカーソル行のファイルを開く、またはフォルダへ移動します。
6. 空行にカーソルを置いて `-` を押すと親フォルダへ移動します。

## コマンド

| コマンド | 説明 |
| --- | --- |
| 現在のフォルダをOilで開く | アクティブファイルのあるフォルダをOil Explorerで開きます |
| Vimキーバインドの切り替え（自動 → ON → OFF） | Oil Explorer内でのVimキーバインドの有効/無効を切り替えます |

## インストール

### 手動インストール

1. [Releases](../../releases) から `main.js` / `manifest.json` / `styles.css` をダウンロードします。
2. Vault の `.obsidian/plugins/oil-explorer/` フォルダを作成し、そこに配置します。
3. Obsidian の設定 → コミュニティプラグイン からOil Explorerを有効化します。

### 開発環境でのビルド

```bash
npm install
npm run dev    # 開発ビルド（ファイル監視）
npm run build  # 本番ビルド
```

## 動作環境

- Obsidian `0.15.0` 以上
- デスクトップ・モバイル両対応
